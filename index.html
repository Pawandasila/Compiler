📚 Custom Language Guide

🔤 Data Types

  • int - Integer values (e.g., 5, -10, 100)
  • float - Floating-point values (e.g., 3.14, -2.5)

📝 Variable Declaration

int x = 5; float pi = 3.14159; int result = x * 2;

🧮 Arithmetic Operations

  • Addition: x + y
  • Subtraction: x - y
  • Multiplication: x * y
  • Division: x / y (integer division for int types)
int a = 10; int b = 3; int sum = a + b; // 13 int difference = a - b; // 7 int product = a * b; // 30 int quotient = a / b; // 3 (integer division)

📤 Output

  • The last expression in your code becomes the output
  • No explicit print statement needed
int x = 5; int y = 10; x + y; // Output: 15

💬 Comments & Syntax

  • Single-line comments: // This is a comment
  • All statements must end with semicolon ;
  • Variables must be declared before use

❌ Not Supported

  • C++ syntax (#include, using namespace, main())
  • Standard library functions from other languages
  • Class definitions or object-oriented features
  • Control structures (if, while, for)
  • Function definitions

✨ Complete Example

// Calculate area of rectangle int length = 12; int width = 8; int area = length * width; // Calculate perimeter int perimeter = 2 * (length + width); // Output the area area;
