Custom Language Guide

Data Types

  • int - Integer values
  • float - Floating-point values

Variable Declaration

  • All variables must be declared with a type
  • Variables can be initialized when declared
int x = 5; float y = 3.14;

Expressions

  • Supports basic arithmetic: +, -, *, /
  • Expressions can be nested with parentheses
  • Supports variable assignments
int result = (x + y) * 10;

Statements

  • All statements must end with a semicolon ;
  • The value of the last expression is returned as output

Comments

  • Single-line comments start with //
  • Block comments use /* */
// This is a comment /* This is a multi-line comment */
